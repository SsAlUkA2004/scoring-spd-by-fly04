# Scoring-Speedrifter-By-Fly04
เว็บนี้ถูกสร้างขึ้นมาเพื่อนับคะแนนในเกม โดยเป็นการนับคะแนนแบบกลุ่ม โดยข้อมูลแต่ละคนแยกกันไปตามอุปกรณ์ 
วิธีการใช้งาน
1.กดเพิ่ม/ลดผู้เล่นตามการแข่งขันที่ต้องการ
2.พิมพ์ชื่อผู้เล่น ใส่ตามช่อง
3.เลือกด่านที่ต้องการ
4.ลงเวลาตามที่เล่นได้ เช่น 11256 ระบบจะประมวลผลเป็น 01:12:56
5.ถ้าต้องการเพิ่มตารางการแข่ง ให้กดปุ่ม "+เพิ่มการแข่งขัน"
ภาษาที่ Support 5 ภาษา(ไม่รวมชื่อด่าน)
Thai,Eng,Vn,Taiwan,China

This website was created to keep score in a game. It uses a group scoring system, but each person's data is tracked separately on their own device.
Instructions
1.Click to add or remove players according to the desired competition.
2.Type the players' names into the provided fields.
3.Select the desired stage.
4.Enter the recorded time (e.g., type 11256, and the system will process it as 01:12:56).
5.To add a competition schedule, click the "+Add Competition" button.
Supported Languages (5 languages, excluding stage names)
Thai, English, Vietnamese, Taiwanese, Chinese
