## 📁 หมวดหมู่ (Category)

ใช้สำหรับจัดกลุ่มการแข่งขัน เช่น รอบแรก / รอบชิงชนะเลิศ

### สร้างหมวดหมู่

1. กดปุ่ม **📁 หมวดหมู่** ที่ด้านบน
2. พิมพ์ชื่อหมวดหมู่ → กด **บันทึก**

### จัดการหมวดหมู่

| ปุ่ม | หน้าที่ |
|------|---------|
| `＋ การแข่งขัน` | เพิ่มตารางใหม่เข้าในหมวดหมู่นี้ |
| `⚡ เวลาด่าน` | ดูเวลาดีที่สุดของทุกด่านในหมวดหมู่นี้ |
| `🏆 คะแนน` | ดูคะแนนรวมของผู้เล่นในหมวดหมู่นี้ |
| `📊 Export` | Export เฉพาะหมวดหมู่นี้เป็นไฟล์ Excel |
| `✏️` | แก้ไขชื่อหมวดหมู่ |
| `🗑` | ลบหมวดหมู่ (ตารางจะถูกย้ายออก ไม่ถูกลบ) |

### ย้ายตารางเข้าหมวดหมู่

ในแต่ละตาราง กดปุ่ม **ชื่อหมวดหมู่** (มีจุดสีวงกลม) → เลือกหมวดหมู่ที่ต้องการ

## 📊 ตารางการแข่งขัน

### เพิ่มตาราง

- กดปุ่ม **＋ เพิ่มการแข่งขัน** ที่ด้านบน

### ปรับจำนวนผู้เล่น

- กดปุ่ม **＋ / −** ในแถบหัวของตาราง
- รองรับ **2–6 ผู้เล่น** ต่อตาราง

### ปรับจำนวนรอบ

- กดปุ่ม **9 รอบ** หรือ **13 รอบ** ในแถบหัวตาราง
- รอบที่ 10–13 จะมีป้าย `NO PT` — ไม่นับคะแนน (บันทึกเวลาได้ปกติ)

### ย่อ/ขยายตาราง

- กดที่ **ชื่อตาราง** (เช่น "การแข่งขัน #1") เพื่อย่อหรือขยาย
- ขณะย่อ จะเห็น **สรุปคะแนนแบบย่อ** ของผู้เล่นทุกคน
- กดปุ่ม **⊟ ย่อทั้งหมด** เพื่อย่อ/ขยายทุกตารางพร้อมกัน

### รีเซ็ตตาราง

- กดปุ่ม 🔄 ที่มุมขวาของตาราง → ยืนยันเพื่อล้างข้อมูลในตารางนั้น

### ลบตาราง

- กดปุ่ม **✕ ลบ** → ยืนยันการลบ (ต้องมีตารางอย่างน้อย 1 ตาราง)

## ⏱ การกรอกเวลา

### รูปแบบเวลา

กรอกเป็นตัวเลขล้วน ระบบจะแปลงให้อัตโนมัติ:

```
15699  →  1:56:99
(นาที : วินาที : เซนติวินาที)
```

หรือกรอกในรูปแบบ `mm:ss:cs` ได้เลย

### การเลือกด่าน (แผนที่)

1. คลิกที่ช่อง **แผนที่** ในแต่ละรอบ
2. พิมพ์ชื่อด่านเพื่อค้นหา หรือเลือกจากรายการ
3. ด่านถูกจัดกลุ่มตามระดับดาว ⭐ (3–7 ดาว)

### สัญลักษณ์ที่ปรากฏในตาราง

| สัญลักษณ์ | ความหมาย |
|-----------|----------|
| 🟡 ขอบสีทอง | เวลาดีที่สุดในรอบนั้น |
| 🔴 ขอบสีแดง | เวลาดีที่สุด **และ** เป็นรอบ KING |
| 👑 KING | รอบพิเศษ — ผู้เล่นที่กำหนดได้คะแนนโบนัสถ้าชนะ |

## 🏆 ระบบคะแนน

คะแนนคำนวณจากส่วนต่างเวลาเทียบกับเวลาดีที่สุดในรอบนั้น

### ตารางคะแนน

| ส่วนต่าง (วินาที) | คะแนน | หมายเหตุ |
|-------------------|-------|----------|
| 0.00 (ดีสุด) | **+11** | — |
| 0.00 + รอบ KING | **+12** | โบนัสพิเศษ |
| < 1.00 | +10 | |
| < 2.00 | +9 | |
| < 3.00 | +8 | |
| < 4.00 | +7 | |
| < 5.00 | +6 | |
| < 6.00 | +5 | |
| < 7.00 | +4 | |
| < 8.00 | +3 | |
| < 9.00 | +2 | |
| < 10.00 | +1 | |
| ≥ 10.00 | +0 | |

### รอบ KING 👑

รอบ KING กำหนดไว้ตายตัวตามลำดับผู้เล่น (P1 → รอบ 1, P2 → รอบ 2, ...):

- ผู้เล่นที่เป็น KING ในรอบนั้น หากทำเวลาดีที่สุด → ได้ **+12** (แทน +11)
- รอบที่ 10–13 **ไม่นับคะแนน** ทุกกรณี

## 📷 การบันทึกรูปภาพ

### บันทึกรูปตาราง (แต่ละ Match)

1. กดปุ่ม **📷** ที่มุมขวาของตาราง
2. ใส่ **หัวข้อ** และ **รายละเอียด** (ไม่บังคับ)
3. กด **📥 บันทึกรูป** → ไฟล์ `.png` จะถูกดาวน์โหลดทันที

รูปที่ได้จะแสดง:
- ชื่อผู้เล่น, ด่าน, เวลา, คะแนนแต่ละรอบ
- เวลาดีสุดไฮไลต์สีทอง/แดง
- แถวคะแนนรวมด้านล่าง

### บันทึกรูปเวลาดีสุดต่อด่าน

1. กดปุ่ม **⚡ เวลาด่านทั้งหมด** (หรือ **⚡ เวลาด่าน** ในหมวดหมู่)
2. ในหน้าต่างที่เปิด → กดปุ่ม **📷 บันทึก**

## 📊 Export / Import Excel

### Export

- กดปุ่ม **📥 Export** (ปุ่มสีเขียว ด้านบน) → Export ทุกตาราง
- หรือกด **📊 Export** ในหมวดหมู่ → Export เฉพาะหมวดหมู่นั้น

ไฟล์ `.xlsx` จะมี:
- ชื่อผู้เล่น, ด่าน, เวลา, คะแนนแต่ละรอบ
- เซลล์ไฮไลต์เวลาดีที่สุดสีเหลือง/แดง
- แถวคะแนนรวมท้ายแต่ละการแข่งขัน

### Import

1. กดปุ่ม **📥 Import**
2. เลือกไฟล์ `.xlsx` ที่ Export มาจากเว็บนี้
3. ระบบถามว่า:
   - **OK** = เพิ่มต่อจากข้อมูลที่มีอยู่
   - **Cancel** = ล้างทั้งหมดแล้วนำเข้าใหม่

> ⚠️ รองรับเฉพาะไฟล์ที่ Export จากเว็บนี้เท่านั้น

## 🔧 ปุ่มและฟีเจอร์อื่น ๆ

| ปุ่ม | หน้าที่ |
|------|---------|
| `⊟ ย่อทั้งหมด` | ย่อ/ขยายทุกตารางพร้อมกัน |
| `⚡ เวลาด่านทั้งหมด` | ดูเวลาดีที่สุดต่อด่านจากทุก Match |
| `🏆 คะแนนรวมทั้งหมด` | ดูอันดับรวมของผู้เล่นทุกคน |
| `🗑 ล้างข้อมูล` | ล้างข้อมูลทั้งหมดและรีเซ็ตหน้าเว็บ |
| `↑` (FAB) | เลื่อนกลับขึ้นด้านบนสุด |

### การบันทึกข้อมูล

- ข้อมูลทั้งหมดบันทึกใน **localStorage** ของเบราว์เซอร์โดยอัตโนมัติ
- ไม่ต้องกดบันทึกเอง — ทุกการเปลี่ยนแปลงถูก save ทันที
- ข้อมูลจะยังคงอยู่เมื่อปิด-เปิดเบราว์เซอร์ใหม่
